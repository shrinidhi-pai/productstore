# 🛍️ Product Store

A modern and responsive **Product Store (E-commerce Website)** built using web technologies. This project allows users to browse products, search items, and view product details with a clean UI.

---

## 🚀 Live Demo
👉 [https://productstore01.netlify.app/]

---

## ✨ Features

- 🛒 Browse products  
- 🔍 Search functionality  
- ❤️ Add to wishlist  
- 📱 Fully responsive design  
- 🎨 Clean and modern UI  
- ⚡ Fast and optimized performance  

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Styling: CSS / Bootstrap 
- Version Control: Git & GitHub
